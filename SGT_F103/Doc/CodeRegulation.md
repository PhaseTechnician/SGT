# 代码规范  
## 函数  
+ 函数名每个单词首字母大写  
+ 层次_模块名_函数功能  
## 变量
+ 首字母不大写，之后每个单词首字母大写  
+ 系统BSP级变量规则单独确立
## 宏定义
+ 值类型宏定义全部字母大写  
+ 函数形宏定义保证有充足的括号来避免意外情况
## 括号和缩进 
```
void BSP_LED_Change(void)
{
	if(output==1)
	{
		BSP_LED_Open();
	}
	else
	{
		BSP_LED_Close();
	}
}
```
